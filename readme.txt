Programa Administrador de Archivos Torrents:

-El programa requiere que el nombre del recurso este compuesto por letras y/o dígitos.

-El programa permite eliminar y editar recursos en el repositorio en  cualquier ubicación.

-Solo se permite insertar recursos adentro de carpetas que no son recursos.

-Cuando hacemos doble click sobre un archivo torrent se ejecutara la aplicación de nuestro cliente 
 torrent favorito, para esto se debe seleccionar el cliente anteriormente.

-El doble click se realiza sobre el archivo torrent, dependiendo de la modalidad de vista del repositorio:
	
	- El listado de archivo: se hace doble click sobre cualquier archivo torrent en la estructura (sobre el treeview).

	- El listado de recursos: se hace doble click sobre un archivo de la lista de archivos torrents asociada a un recurso.
	  La lista es cargada haciendo doble click sobre el recurso correspondiente.
